## Getting Started

### What do I use:
- SCSS
- Webpack + Babel
- Handlebars/Pannini
- Gulp
- Browser Sync
### Install
```
npm run install
```
### Develop
```
npm run start
```
### Build
```
npm run build
```
###` Deploy
Travis CI deploy to GitHub Pages automatically after push to master branch
